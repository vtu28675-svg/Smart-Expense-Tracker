# Smart Expense Tracker with Analytics Dashboard

A professional Python-based expense management application designed to track, analyze, and visualize daily expenses.

## Features
- Add and manage expenses
- Category-wise expense tracking
- Expense summaries and analytics
- CSV-based data storage
- Expense visualization using pie charts

## Technologies Used
- Python
- Pandas
- Matplotlib

## Functionalities
- Record expenses with categories
- Analyze total spending
- Generate category-wise summaries
- Visualize expense distribution

## Project Type
Data Analytics / Finance Management Application
